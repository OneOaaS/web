申请使用流程

温馨提示: 安装可以是Zabbix WEB所在服务器，如果是一台全新服务器，需要能够读写Zabbix DB

使用步骤:

rpm -ivh oneoaas-monitor-ce-linux-amd64-2.0.0-1.x86_64.rpm

vim /usr/share/oneoaas-monitor-ce/conf/app.conf

/etc/init.d/oneoaas-monitor-ce start

一、配置说明：

shell# vim /usr/share/oneoaas-monitor-ce/conf/app.conf

httpport = 4005      #访问端口

dbtype = "mysql"     #zabbix 数据库的类型，默认为MySQL

dbuser = "zabbix"    #zabbix 数据库的用户

dbpass = "zabbix"    #zabbix 数据库的密码

dbhost = "127.0.0.1"   #zabbix 数据库的IP地址

dbport = 3306      #zabbix 数据库的端口

dbname = "zabbix"   #zabbix 数据库的库名称

#zabbix Web的地址，即Web API，Zabbix Web地址为http://10.10.10.38,则配置为ZbxUrl = "http://10.10.10.38"

#带路径的配置，如果您的Zabbix WEB访问地址为http://10.10.10.38/zabbix,则配置为

#ZbxUrl = "http://10.10.10.38/zabbix"

shell# /etc/init.d/oneoaas-monitor-ce start

二、访问地址

http://您的IP地址:4005 用户: 您的Zabbix用户 密码: 您的Zabbix密码

三、手动测试API连接

shell# curl -X POST -H 'Content-Type: application/json' -d '{"jsonrpc": "2.0","method":"user.login","params":{"user":"Admin","password":"zabbix"},"id":0}' http://10.10.10.38/api_jsonrpc.php

输出如下结果，则说明http://10.10.10.38为正确的Zabbix API地址{'jsonrpc':'2.0','result':'b93e7c8387a5c55b64c29f6e7830622e','id':0}
